# Encrypt - Decrypt Terminal (python tool)


tool install command

```
$ git clone https://github.com/userfax1/encrypt-decrypt-ui-terminal.git
```
install pyperclip

```
$ pip install pyperclip
```

tool start command

```
$ python caesar_terminal.py
```

press e to encrypt, d to decrypt, r for random shift
type message, you'll see "Characters: 11"
shift defaults to 2 if you hit Enter
result prints and copies automatically if you installed pyperclip

you need install python !!
